# isometric style

**Vibe.** Glassmorphic 3D panels at 30° angle. Untitled UI / Stripe website / Figma plugin landing page. Premium modern tech.

**Best for.** Product decks, engineering presentations, "this looks like a Series-B SaaS landing page" energy. Hero slides for technical content.

## Visual identity

```
- 30° isometric projection with soft long drop-shadows behind each shape (long, low-opacity, angled to match the projection)
- Each shape becomes a glassmorphic 3D panel — semi-transparent white-to-light-blue gradient surface, subtle depth, 4px rounded corners visible at the front face
- Modern, premium tech aesthetic — clean, airy, lots of negative space around the panels
- Faint dotted-grid background (very subtle, 24px grid, 5% opacity)
- Everything sits flat in the same isometric plane — no varying angles
```

## Icon directives

```
Style of icons: simple line-art icons drawn in isometric perspective with one accent color stroke (gold or coral), ~2px lines. Each icon sits inside its panel with a slight 3D feel — books stacked, laptop open, clipboard tilted at the iso angle. Composed cleanly, not crowded.
```

## Palette

```
Background: warm cream-white (#faf8f4). Panel fill: white-to-light-blue gradient (#ffffff → #dbeafe), semi-transparent with subtle highlight on the top edge. Strokes: navy (#1e3a5f). Accents: soft pastel teal (#5eead4) for arrow ribbons, coral (#f59e8a) for icon highlights, gold (#d4af37) for icon stroke accents. Text: deep navy.
```

## Typography

```
Inter or SF Pro — clean geometric sans-serif. Title in semibold at the top, sized prominently. Labels in regular weight, possibly italic for the playful touch (the verified render used italic labels on the panel front faces — works well with the iso angle).
```

## Background

```
Cream-white with very faint dotted grid (5% opacity, 24px spacing). No texture, no gradient on the canvas itself — just the grid as a subtle anchor.
```

## Special notes

The 30° projection consistently across all panels is the key — if any panel renders flat, the whole composition breaks. The verified render has all 4 panels at the same iso angle with arrow ribbons curving between them in matching 3D perspective.
