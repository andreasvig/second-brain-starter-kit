# clean-vector style

**Vibe.** Modern flat-design — clean 2px vector strokes, Inter typeface, subtle drop shadows, refined palette. Default presentation polish.

**Best for.** Slack posts, internal documentation, default "make this look professional" passes. Safe choice when in doubt.

## Visual identity

```
- Clean 2px vector strokes with subtle anti-aliasing — no roughness, no hand-drawn texture
- Each shape becomes a refined rectangle (or ellipse / diamond as in source) with 4px rounded corners
- Subtle drop shadow on each shape: 2px y-offset, 4px blur, 12% opacity black
- Plenty of negative space, modern flat-design aesthetic
- No gradients, no glows, no decorative elements
```

## Icon directives

```
Style of icons: simple flat-design line icons in a single accent stroke color, ~2px line weight, geometric and minimal — Lucide / Heroicons style. One icon per shape, centered above or beside the label, sized at roughly 30–40% of the shape's smaller dimension.
```

## Palette

```
Background: clean white. Shape fills: very pale tint matching each role (soft blue for core, soft purple for tool, soft green for storage, soft red for output, white for trigger/neutral). Strokes: dark navy (#1e293b). Accent (only for the central / core element): muted blue (#3b82f6). Text: dark slate (#0f172a).
```

## Typography

```
Inter or a similar modern geometric sans-serif (SF Pro, IBM Plex Sans). Title in 1.5× the label size, semibold, dark slate. Labels in regular weight, same color. No serifs anywhere.
```

## Background

```
Pure clean white. No texture, no gradient, no grid.
```
