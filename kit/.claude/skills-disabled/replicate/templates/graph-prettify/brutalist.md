# brutalist style

**Vibe.** 1980s Swiss design / editorial print spread — Wim Crouwel, Helmut Schmid, early Total Design. Stark newsprint cream, heavy black rules, massive condensed display sans-serif, woodcut silhouette icons, single signal-red accent.

**Best for.** Editorial content, bold statements, magazine-spread vibes, talks where the user wants the diagram to feel "this is a manifesto, not a slide". Strong personality — not for everything.

## Visual identity

```
- Stark newsprint-cream background with subtle paper / printing texture
- Heavy black rules (4–6px) and pure geometric rectangles — NO rounded corners, no shadows, no gradients
- Strong pale-gray grid visible in the background (10% opacity, ~24px spacing) — sells the editorial feel
- Each stage numbered prominently (01, 02, 03, 04) in a massive condensed display sans-serif above or inside each shape
- Heavy black filled-triangle arrows between shapes — simple and confident
- Optional: a thick horizontal black rule under the title for that magazine-feature feel
```

## Icon directives

```
Style of icons: woodcut-style or stencilled silhouette in pure solid black, no inner detail beyond what's needed for legibility. Bold, graphic, instantly readable at small size. Think 1970s printing — magnifying glass over papers, classic typewriter, checkmark inside a square, vintage printing-press wheel or megaphone silhouette. Each icon centered above its label inside the box.
```

## Palette

```
STRICT three colors only:
- Newsprint cream (#f4f1e8) — background
- Pure black (#000000) — rules, type, icons, arrows, box outlines
- Signal red (#dc2626) — used SPARINGLY: only for the rule under the title, OR one icon for emphasis, never for fills

Do not introduce a fourth color. The constraint is the style.
```

## Typography

```
Massive bold condensed display sans-serif for the title — Akzidenz Grotesk Bold Condensed / Helvetica Inserat / Druk Wide / similar industrial workhorse. ALL CAPS for the title, anchored top-left. Stage numbers (01, 02, 03, 04) in the same family, even larger relative to box height. Labels in a slightly less aggressive weight (regular or medium condensed sans), all caps or small caps. Everything pure black.
```

## Background

```
Newsprint cream with subtle paper texture (very fine grain), and a pale-gray grid (10% opacity, 24px spacing) visible behind everything. The grid is the editorial-design signature — without it the layout feels floating.
```

## Special notes

The single accent red is the entire color personality of this style — overuse it and the brutalism breaks. The verified render uses red on exactly one element (the rule under the title) and nothing else. The grid background is also load-bearing — it's what makes the layout read as "designed" rather than "diagrammed".
