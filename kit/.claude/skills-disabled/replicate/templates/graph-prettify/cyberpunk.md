# cyberpunk style

**Vibe.** Futuristic operations dashboard. Blade Runner 2049 / Cyberpunk 2077 UI panel. Deep navy/black background, neon-bordered panels with chromatic aberration, glowing wireframe icons, scanlines.

**Best for.** YouTube AI content, futuristic / tech / agentic narratives, anything where the user wants high visual impact and "this is the future" energy.

**Note on role coding.** nano-banana-pro respects explicit role coding when stated in the INTENT — e.g. cyan borders on hub + knowledge-store spokes, magenta on operational tool spokes. Useful when the diagram has semantic node categories worth distinguishing.

## Visual identity

```
- Deep dark-navy / near-black background with a faint glowing grid overlay (5% opacity, ~32px spacing)
- Each shape is a neon-bordered glassmorphic panel with subtle chromatic aberration on the edges (slight cyan / magenta fringing)
- Interior of each panel is dark and semi-transparent — the panel reads as a piece of glass mounted in front of the dark background
- Faint horizontal scanlines and tasteful digital noise across the whole image (low intensity — atmosphere, not distraction)
- Slight bloom around bright elements — borders, icons, arrowheads
- Optional: small UI chrome details in corners (tick marks, bracket frames, "STAGE 01" microtext) for that operations-panel feel
```

## Icon directives

```
Style of icons: glowing neon wireframe icons in cyan or magenta, ~1.5px line weight, with subtle bloom. Holographic feel — semi-transparent, slightly translucent. Each icon is centered inside its panel with negative space around it. Think HUD elements: holographic magnifier scanning a wireframe document, glowing keyboard with code streams, digital eye with checkmark, satellite uplink beaming a packet.
```

## Palette

```
Background: deep navy / near-black (#0a0e1a). Panel borders: electric cyan (#00f0ff) on some, neon magenta (#ff007f) on others — alternating or matched to shape role. Interior fill: very dark navy semi-transparent (#0a0e1a at 60% over the background grid). Text: off-white (#f0f4ff) with subtle cyan glow. Accent highlights: neon lime (#a8ff00) used sparingly for emphasis. Arrows: glowing cyan with bloom.
```

## Typography

```
Thin tech sans-serif or monospace — Eurostile / Orbitron / IBM Plex Mono / SF Mono. Title at top in light or thin weight with a soft glow. Labels in a similar weight, all caps optional. Microtext (e.g. "STAGE 01") in mono, smaller, lower opacity. Avoid heavy or rounded fonts — the cyberpunk look is thin and angular.
```

## Background

```
Deep navy/black canvas with a faint glowing grid (5% opacity cyan, 32px spacing). Add subtle horizontal scanlines (very low opacity) and a touch of digital noise. No texture beyond that — the grid + scanlines do the work.
```

## Special notes

The chromatic aberration on panel edges is the cyberpunk signature — small cyan/magenta fringe ~1px on the borders. Without it, the style reads as "neon" not "cyberpunk". Also keep the scanlines subtle — too aggressive and the diagram becomes unreadable. The verified render nails this balance.
