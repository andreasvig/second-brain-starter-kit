# storybook style

**Vibe.** Children's-storybook illustration in watercolor. Beatrix Potter meets Studio Ghibli. Soft, dreamy, organic shapes (no rigid rectangles), whimsical animal characters acting out each stage.

**Best for.** Whimsical explainers, onboarding content, family / non-technical audiences, talks where the user wants warmth and personality. Strong personality — not for serious technical content.

## Visual identity

```
- Soft watercolor paper background with visible texture and a gentle warm gradient (lighter in the center, slightly darker at the edges)
- Each shape becomes an organic, hand-painted rounded cloud-shape (NOT a rigid rectangle) with watercolor bleeds at the edges
- Dreamy, gentle lighting — soft but not foggy
- Dark muted sienna ink outlines, graceful and slightly imperfect
- Tiny details — flowers, leaves, stars, dots — scattered in the negative space (subtle, not crowding)
- Everything reads as a single illustrated scene, not a diagram with decoration on top
```

## Icon directives

```
Style of icons: whimsical watercolor character illustrations — small animal characters acting out each stage. Mouse reading a book in a cozy library nook with stacked books / fox at a wooden writing desk with quill and inkwell / wise spectacled owl examining a scroll / paper boat sailing on watercolor waves toward a sunrise. Each character is the icon — they bring the workflow to life. Painted in the same soft watercolor style as the shapes, with the same sienna ink outlines.
```

## Palette

```
Background: cream paper (#fef9f0) with warm gradient. Watercolor washes in soft, cohesive tones: sage green (#a8b89c), dusty rose (#e0a8a8), sky blue (#a8c8e0), butter yellow (#f0d878), warm sienna (#a06848) for outlines. Each shape gets one wash + character composition. The palette feels like a vintage children's-book plate — muted, warm, gentle.
```

## Typography

```
Hand-lettered or whimsical serif — think Pacifico / Caveat / a soft display serif with slight personality. Title at top in a flowing hand with confidence (not childish — refined storybook). Labels under or near each character/shape in the same hand, slightly smaller. Everything in the warm sienna ink color, not pure black.
```

## Background

```
Cream watercolor paper with a gentle warm gradient (lighter center, slightly darker edges — like an old book page). Tiny scattered florals and leaves in the negative space between shapes — sparse, atmospheric. The arrows between shapes can become dashed-line "paths" or trails of dots, like footprints between scenes.
```

## Special notes

The organic cloud-shapes (instead of rigid rectangles) are the signature — they make the whole composition feel like an illustrated scene. The shape-arrangement still preserves the original layout (left-to-right or whatever the source has), but each "shape" becomes a softly painted bubble of scenery rather than a box. Without this, the style reads as "watercolor on top of a diagram" rather than "diagram-as-illustration".

Risk: this style has the loosest preservation of the six. If the source diagram has tight precise structure (lots of arrows, dense connections), test on `clean-vector` first and step up to storybook only if the user wants the whimsical treatment. For simple sequential / hub-and-spoke layouts (like pipeline), it works beautifully.
