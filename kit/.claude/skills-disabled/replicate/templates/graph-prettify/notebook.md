# notebook style

**Vibe.** Hand-illustrated Moleskine entry. Refined bullet-journal aesthetic — watercolor washes, ink line work, hand-lettered text. Warm, personal, considered. Not childish — adult sketchbook.

**Best for.** Personal reflections, storytelling content, casual blog or LinkedIn posts where the user wants the diagram to feel "drawn while thinking, not printed in a deck". Editorial content with a human touch.

## Visual identity

```
- Cream paper background with subtle grain and very faint ruled lines (visible but unobtrusive)
- Hand-drawn ink boxes with refined slightly-imperfect lines — corners not quite square, lines slightly varying in weight, but graceful, not messy
- Soft watercolor washes inside each shape, transparent and slightly bleeding past the ink edges (the bleed is the signature)
- Hand-lettered labels in a careful sketch script — readable, considered, not childish
- Tiny ink details in negative space (dots, dashes, small flourishes) for texture
```

## Icon directives

```
Style of icons: whimsical doodles in the same dark sepia ink as the boxes, with the same watercolor wash treatment — slightly imperfect, charming, hand-rendered. Each icon sits inside its box with the watercolor wash bleeding around it. Think bullet journal aesthetic — small stack of books, fountain pen, checklist, scroll.
```

## Palette

```
Cream paper (#fef9f0) with subtle grain. Ink: dark sepia (#3d2914) — never pure black, always warm. Watercolor washes in muted earth tones: ochre (#c8a064), sage green (#8a9a7b), dusty rose (#d4a3a0), soft slate (#7a8b9e). Each shape gets one wash color (rotate through the palette). Hand-lettered text in the same dark sepia ink.
```

## Typography

```
Hand-lettered script — refined, not childish. Bullet-journal aesthetic: title in a flowing slightly-italic hand with a subtle underline flourish; labels in a careful sketched script slightly thicker than the title. Use the same ink color throughout. Avoid digital fonts — everything looks drawn.
```

## Background

```
Cream paper with subtle grain and very faint horizontal ruled lines (5% opacity, evenly spaced). The paper edges may have very slight darkening for warmth. No frame, no border decoration.
```

## Special notes

The watercolor bleed past the ink lines is what sells the style — without it, it looks like a flat sketch. Make sure the prompt asks for the bleed explicitly. Also: keep ink one consistent dark sepia tone, not varying — that's what unifies the page.
